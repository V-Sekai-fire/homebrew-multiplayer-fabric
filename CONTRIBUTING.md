# Contributing

A Homebrew tap that distributes multiplayer-fabric CLI tools on macOS.
Formulae live in `Formula/` and are evaluated by `brew install
multiplayer-fabric/<tap>/<formula>`.  No build system beyond Homebrew
itself — contributions are formula edits and version bumps.

## Guiding principles

- **One formula per tool.** Each binary or CLI gets its own `.rb` file.
  Shared logic belongs in a helper module, not copy-pasted across
  formulae.
- **Pin the SHA256.** Every `url` must have a matching `sha256`.
  Never leave it blank or use a placeholder; generate with
  `shasum -a 256 <downloaded-tarball>`.
- **Audit before submitting.** Run `brew audit --strict Formula/<name>.rb`
  and `brew install --build-from-source Formula/<name>.rb` locally.
  Both must pass without warnings.
- **Test blocks are required.** Every formula must include a `test do`
  block that invokes the installed binary and checks its exit code or
  output.  A test that just runs `--version` is acceptable if no
  command-line operation is side-effect-free.
- **Keep bottles out of PRs.** Bottle hashes are generated by CI.
  Do not add or modify `bottle do` blocks manually.

## Version bump workflow

1. Download the new release archive and compute its SHA256.
2. Update `url` and `sha256` in the formula.
3. Remove any `revision` line unless a formula-level patch has been
   re-applied.
4. Run `brew audit --strict` and `brew install --build-from-source`.
5. Commit as `Bump <name> to <version>`.

## Design notes

### Formula vs cask

These formulae install compiled CLI binaries (Elixir escripts or native
executables).  Avoid `cask` — it implies a GUI app.  If a release
ships a `.app` bundle in the future, that belongs in a separate cask tap.

### Vendored archives

Upstream releases are hosted on GitHub.  Prefer the `.tar.gz` asset
over the auto-generated source archive; the former contains only the
binary and is stable across re-tags.  If the upstream publishes
platform-specific binaries, use `on_macos` / `on_arm` guards to select
the correct archive per architecture.
